Pandas er et bibliotek til at arbejde med data.

Man kan blandt andet:
- Importere datafiler. .csv, .xls eller json f.eks.
- Analysere data. Gennemsnit, totaler osv.
- Cleane data: 
	- Omdøbe kolonner, rækker, fjerne data på programmerings niveau.
- Gøre data klar til visning i UI eller klargøre til ML med *scikit-learn* f.eks
