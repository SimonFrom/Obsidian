[Tuple](https://www.tutorialsteacher.com/csharp/csharp-tuple)
 
- En tuple er datastruktur der kan indeholde flere forskellige typer af data.
- Tuple\<T1,T2,T3….T7\>
    - Tuple\<int, string, string, float\> = new Tuple\<int, string, string, float\>(1, "Hej", "Med", 2.89)
- En tuple kan maks indehold 8 elementer.
- Det er en reference type og ikke en værdi type
