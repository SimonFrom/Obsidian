- En value type overskriver den gamle værdi med en ny.
- En reference type gemmer en reference af den originale og overskriver ikke.
 
- Med lister kan man gemme værdier, lidt som arrays, men mere fleksibelt.
- List\<\> kan man f.eks tilføje og fjerne elementer.
- Med Dictionary\<\> kan man "kode" en værdi sammen med en key/nøgleord.
 
- Med method overloading kan man have to metoder med samme navn, men forskelligt indhold.
