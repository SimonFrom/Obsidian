git init:

- Bruges til at inisialisere et repository

git add:

- Bruges til at tilføje en fil til køen i git

git commit:

- Bruges til at sende filen til git

git status:

- Bruges til at printe en liste over sine addede filer og commits der venter

git clone --branch (branchname) (remote-repo-url)

- Bruges til at clone en specifik branch, indsæt branch name og url på main branch. Paranteser skal ikke med i git!!!

Git checkout

- Kan bruges til at restore en tidligere version eller flytte over på en anden gren/branch.
