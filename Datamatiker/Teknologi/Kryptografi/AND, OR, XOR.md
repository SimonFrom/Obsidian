# AND:

I et AND statement kan skifte AND ud med et gange tegn f.eks.
Så:
0 AND 0 = 0
1 AND 0 = 0
1 AND 1 = 1

Så med et binært tal ville det se således ud:

100111001011010100111010 AND 010110100001101111011000 = 000110000001000100011000

Altså tager man hvert tal og sammenligner med det på den anden side af AND.
Et AND'et tal kan ikke blive størrere end de to forrige tal

# OR:

OR outputter 1 når en eller flere af inputs er 1.
0 OR 0 = 0
1 OR 0 = 1
0 OR 1 = 1
1 OR 1 = 1

Resultatet af et OR kan ikke blive mindre end de 2 oprindelige.

# XOR:

XOR outputter 1 når inputs ikke matcher:

0 XOR 0 = 0
0 XOR 1 = 1
1 XOR 0 = 1
1 XOR 1 = 0

Resultatet af XOR kan være en hver tænkelig kombination.
