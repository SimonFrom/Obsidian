Layer 3 device fordi den opererer med IP adresser.

*En router kan beskrives som en enhed der kan sende data fra et netværk til et andet. Fra den private ip adresse til den offentlige f.eks.*

Routing table -  Et kort over alle kendte netværk ruter som routeren har været i kontakt med. Starter tomt og bliver fyldt efterhånden.

Består af:

| Navn på netværk | **IP adresse** | Connection type/(Static eller Directly Connected) |
| --------------- | -------------- | ------------------------------------------------- |

Static route - Manuelt konfigureret netværks rute.

Directly Connected - Når routeren bliver konfigureret med en specifik IP adresse ved den hvor de forskellige netværk ligger
